# Unity Runtime

This Unity project lives under `apps/unity-runtime` and uses Unity `6000.4.1f1`.

Current implementation: this is the live Unity runtime source of truth for room bootstrap, avatar presentation, bridge wiring, audio playback, and runtime-focused tests.

Current scope:

- `Assets/Scripts/App/`
- `Assets/Scripts/Runtime/`
- `Assets/Scripts/Audio/`
- `Assets/Scripts/Avatar/`
- `Assets/AvatarSystem/`
- `Assets/Tests/`

Legacy Unity business UI and UI Toolkit shell code are not part of this project anymore.
