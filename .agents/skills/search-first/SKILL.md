# Search First

Research the repo and available capabilities before adding new automation code.

## Triggers

- starting a new feature
- considering a new tool or dependency
- before creating a new helper or adapter

## Checklist

- search the repo for an existing implementation
- check the canonical catalog before creating new harness surfaces
- prefer extension or reuse over net-new duplication

## Outputs

- repo findings
- reuse decision
- implementation direction
