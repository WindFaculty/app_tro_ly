# Continuous Learning

Capture reusable lessons from workflow runs and refactors.

## Triggers

- after verification identifies a repeatable pattern
- after a workflow recovery path succeeds
- after a structural refactor lands

## Checklist

- record the smallest reusable lesson
- attach evidence from logs or tests
- prefer durable patterns over one-off anecdotes

## Outputs

- lessons
- reusable patterns
- future guardrails
